### Java' s Data Type
数据类型Data type是指一组值和这组值上的操作. 如int指所有整数和在这些整数上的加减乘除运算操作, 运算结果的数据类型和参与预算的数据类型相同, 所以5/3等于1, 而5.0/3.0等于1.6666666.... 当1+2.5时, 1会被提升为double类.
